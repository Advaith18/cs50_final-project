Best linux distros for users who experienced linux :

People like me who have used distros will really love these distros. They are very stable environment will rolling updates frequently.
This distros do have a huge community because of their perfomance and power. Now these distros will be a little tricky to figure out if you are a fairly new user.
Although these distro are the ones which people dream for if they know one or two things about linux.




These are the distros in my prospective which are good for a little experienced users:

1. Garuda Linux

2. Manjaro Linux

3. Fedora Linux

1. Garuda Linux

This is one of the best looking distro. It is quite user friendly and perfomance oriented. It uses a btfrs file system and is available in
lot of desktop environments. It is a arch based linux distro and uses the pacman package manager. There is also an edition with some blackarch tools for penetration testing. It is very simple to install and use.

2. Manjaro Linux

Manjaro is a powerful distro which is Arch based Linux which uses a pacman package manager. It is very low on system resources and has a huge community.
There are some desktop environments which are community driven and there are ARM editions also. It is available in 3 desktop environments :
1. KDE plasma
2. XFCE
3. Gnome

3. Fedora Linux

Fedora is a good looking and powerful distro which is redhat based distro and uses a DNF package manager. It is a gnome based distro , the all new fedora workstation 36
uses the gnome 41. It is well suited for developers. Even the inventer of Linux Linus Torvards uses fedora workstation on his macbook.
