The fastest Linux distros are :

The given distros are very good to revive old PC's or for a PC where performance is key and it has to be very low on system resources.
All these distros aim to be resource efficient and fast. They are also very good to set up virtual machine's as they are low on resources.
They may not come with a lot of pre installed software but are very snappy. I would not suggest a new user to try any of these because you will have to install the packages manually.



These are the distro which are fast and usable :


1. Linux Lite

2. Lubuntu

3. Zorin OS lite

1. Linux Lite

Linux lite. For. Me it is the best Linux distro in terms of:

Lightness ( the name itself says that!), Performance/speed (blazing fast) Amazingly Beautiful UI even alongside less resource consumption (particularly Linux lite 4.2), Beginner friendly, Old /ancient desktop reviver
It uses a XFCE desktop environment by default with few changes which makes it power efficient as well as awesome looking.

2. Lubuntu

Lubuntu is distro based on Ubuntu which uses APT package manager and aims to provide users with a desktop which is very fast. It uses Lightweight X11 Desktop Environment (LXDE) as the default desktop environment. Lubuntu is a lightweight variant of Ubuntu and a great choice for beginners looking to give an old computer a second lease on life.
Lubuntu is a great distro for breathing new life into an old computer.

3. Zorin OS Lite

Zorin OS Lite is a Ubuntu based distro which uses the APT package manager. It uses an XFCE as the dafault desktop environment.
Performance wise, I found everything fast, even when running multiple apps at a time with numerous tabs open in Firefox. It just works well and I highly suggest giving it a try if you love XFCE.
The zorin os team have made zorin os very simple to use and install. The zorin appearence app allows you to customize the desktop layout easily.
The zorin connect app allows the user to connect to their respective phone easily.
